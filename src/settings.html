Metafields

Add any custom metafields that you want to sort and/or filter on. If the user's javascript is enabled, they will appear, otherwise sorting will default to Shopify's native default options. By specifying "ui component", one can choose the appearance and function of the sorting, a list and explanation is provided below:

  • Selectbox
  • Mult. Selectbox
  • Radio Buttons
  • Checkbox

Tags

Any tags added to a product with the naming scheme kvp:key:value can be used to create sorting and filtering options. For example, 2 products with the tags kvp:color:black and kvp:color:red will generate a selectbox called Color with the two selections black and red to filter the products in any given collection with both. In addition to adding the tags kvp:key:value, one must list the name of the key below to register it as a filter option. Different UI elements can be chosen in a similar way to MetaFields above.

Price

Add a price range filter. The collections page will read all the prices of its products and divy up the lowest price and highest price into the amount of options specified in "Range Splits". The idea here is that on your Hard Drives collection, if your cheapest hard drive is $40 and your most expensive $470, and you say you want to split the price range by 5, you will get a set of options like so:
$40 - $149, $150 - $249, $250 - $349, $300 - $399, $400 - $470
Depending on the "UI Component" you choose, these will display differently. Checkboxes will list out each one of these from top to bottom, where you can select one or many. Range Slider is very different, where you will have the lowest price on the far left of a slider, and the highest on the far right. There will be notches on the slider which correspond to each price point, and two points form a band which can snap on or around those notches, denoting the range which will show.

Custom Icons and Images

For any tag or metafield, you can provide a custom image, icon code (this theme uses icomoon), or color value to display alongside each value of the filter UI component. For example, if you have a "Color" filter option set, you can specify each color black, blue, green to a hex code, and your select-box will change accordingly.
